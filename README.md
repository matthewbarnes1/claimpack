# ClaimPack

ClaimPack — carrier-ready claim documentation for restoration contractors.

Part of a five-team AI-industry codeathon (July 2026). Whitepaper served live at `/whitepaper`.

## Run

```bash
npm install && npm start   # http://localhost:3015
```

Node >= 22.5, zero native dependencies. Deploys to Vercel out of the box. Demo analysis engines are deterministic; production connects an LLM API key. MIT licensed.
